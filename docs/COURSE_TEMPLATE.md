# 课程内容模板

> 使用此模板快速创建新课程内容，复制后替换对应内容即可。

---

## 模板结构

```json
{
  "stages": [
    {
      "id": "stage-1",
      "title": "第一阶段：入门基础",
      "totalLessons": 6,
      "totalHours": 6,
      "background": "介绍本阶段的背景和学习目标，让学员了解将要学什么、为什么学。",
      "objectives": [
        "掌握XXX基础知识",
        "了解YYY基本概念",
        "完成ZZZ实操任务"
      ],
      "lessons": []
    }
  ]
}
```

## 课时类型详解

| type值 | 说明 | agentMode推荐 | aiEnabled |
|--------|------|---------------|-----------|
| `content-reading` | 理论内容阅读 | `knowledge` | `false` |
| `practice` | 实操练习 | `socratic` | `true` |
| `exam` | 阶段测试 | `examiner` | `false` |

## 快速创建新课程

### 步骤1: 复制模板
复制上述JSON模板到新文件

### 步骤2: 修改基本信息
- 修改 `id` 为新课程ID
- 修改 `title` 为课程名称
- 修改 `background` 为课程介绍

### 步骤3: 设计学习阶段
- 确定分为几个阶段
- 每个阶段包含几个课时
- 设计阶段难度递进

### 步骤4: 编写课时内容
- 按模板格式编写每个课时
- 设置正确的课时类型
- 准备对应的Markdown内容

### 步骤5: 测试验证
- 在平台上导入新课程
- 测试各类型课时的功能
- 验证AI对话模式
